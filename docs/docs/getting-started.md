# Getting Started

This page walks you through what you need before using AutoNyan and how to add
your first document.

## Before you begin

You need just two things:

- **Access to the watched Drive folder.** Whoever set up AutoNyan shares a
  Google Drive folder with you — this is where you drop documents to be
  processed. If you cannot find it, ask the person who runs your AutoNyan
  instance to share it with you.
- **An email address for notifications.** AutoNyan emails you a summary after it
  processes your documents. This is configured by whoever runs the instance.

You do **not** need to install anything or sign in to any new service. If you can
use Google Drive, you can use AutoNyan.

## Set up your categories

AutoNyan files each document into a category, and the categories are just the
folders inside the category area of Drive. Before adding documents, make sure a
few category folders exist — for example:

- `Invoices`
- `Receipts`
- `Contracts`

Create one folder per category you want. You can add or rename categories at any
time; the next document AutoNyan processes will use the updated list. If no
category fits a document, AutoNyan files it under **Uncategorized**.

## Add your first document

1. Open the watched Drive folder that was shared with you.
2. Drag a document into it (or upload it the way you normally would). See
   [Supported File Types](./supported-files.md) for which kinds of files
   AutoNyan can read.
3. Wait. AutoNyan checks the folder on a schedule, so processing is not
   instant — give it a little time.

## Where your results appear

Once a document has been processed:

- It **moves out** of the watched folder and into the matching **category
  folder** (or **Uncategorized** if nothing fit).
- You receive a **notification email** summarizing what was processed, the
  category it was filed under, and a short summary of its contents.

If a document stays in the watched folder longer than you expect, that is usually
just timing — AutoNyan scans on a schedule rather than the moment you drop a
file.
