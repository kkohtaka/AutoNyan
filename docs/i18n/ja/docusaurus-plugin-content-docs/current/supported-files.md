# 対応ファイル形式

AutoNyan は、書類を分類・整理するために中のテキストを読み取ります。このページ
では、最初から最後まで一通り処理できるファイル形式を説明します。

## 完全に対応

次の書類は、自動で読み取り・分類・整理されます。

- **PDF**（`.pdf`） — 紙の書類をスキャンまたは撮影して PDF として保存したものも
  含みます。ページが紙の画像であっても、PDF 内のテキストが抽出されます。
- **プレーンテキスト**（`.txt`）。

最良の結果を得るには、監視対象フォルダに置く前に書類を **PDF として保存または
書き出す**ことをおすすめします。

## 検出はされるが、まだ読み取れない

次の形式は現在、AutoNyan が*検出*してコピーはしますが、テキストをまだ抽出でき
ません。そのため**分類も整理もされず**、内容の要約も得られません。

- **Microsoft Office** — Word（`.doc`, `.docx`）、Excel（`.xls`, `.xlsx`）、
  PowerPoint（`.ppt`, `.pptx`）。
- **リッチテキスト**（`.rtf`）。
- **Google Workspace** — Google ドキュメント、スプレッドシート、スライド。

現時点でこれらを処理するには、まず **PDF として書き出して**ください（Google
ドキュメント: *ファイル → ダウンロード → PDF*、Microsoft Office: *名前を付けて
保存 → PDF*）。その PDF をフォルダに置きます。

## 画像単体について

写真やスクリーンショットなどの画像ファイル単体（`.jpg`、`.png` など）は、監視
対象フォルダからは**そのままでは検出されません**。紙の書類を撮影したものを処理
するには、まず **PDF として保存**してください。そうすれば、上記のとおりテキスト
が抽出されます。

:::note

このページは現在の AutoNyan の動作を反映しています。迷ったときは、必ず機能する
PDF 形式を選んでください。

:::
