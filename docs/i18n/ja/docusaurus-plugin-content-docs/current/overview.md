# AutoNyan のしくみ

AutoNyan は、共有された 1 つの Google Drive フォルダを「自動で整理される受信
箱」に変えます。書類を置くだけで、AutoNyan が中身を読み取り、置き場所を判断し、
ファイルし、何をしたかを知らせます。すべて自動で行われ、操作は不要です。

## 全体の流れ

監視対象のフォルダに書類を追加すると、次の 5 つのステップを通ります。

1. **検出** — AutoNyan は監視対象のフォルダを定期的に確認し、新しく追加された
   書類を見つけます。
2. **読み取り** — 各書類の中のテキストを抽出します。紙の書類をスキャンまたは
   撮影し、PDF として保存したものも対象です。
3. **分類** — 抽出したテキストをあなたのカテゴリと照合し、最も合うものを選び
   ます。
4. **整理** — 書類を該当するカテゴリフォルダへ移動し、そのフォルダの既存
   ファイルの命名に合わせた、内容がわかる名前にリネームします。うまく合う
   ものがなければ **未分類（Uncategorized）** フォルダへ元の名前のまま移す
   ので、書類が失われることはありません。
5. **通知** — 何が処理され、各書類がどのカテゴリに入り、内容がどのようなもの
   だったかをまとめたメールが届きます。

> **ファイルを置く → 検出 → 読み取り → 分類 → 整理 → メールが届く**

## カテゴリはあなたが決める

AutoNyan が勝手にカテゴリを作ることはありません。カテゴリとは、Drive のカテゴリ
領域内に**あなたが作成したフォルダ**そのものです。たとえば `請求書`、`領収書`、
`契約書` などです。AutoNyan はフォルダ名を読み取り、各書類を合うものに振り分け
ます。新しいカテゴリを追加するにはフォルダを作り、不要になったら削除します。
整理の仕方はあなたが管理します。

## 変わらないこと

- これまでどおり Google Drive を使うだけです。
- 手作業で書類を振り分ける必要はありません。
- 元の書類は削除されず移動されるだけなので、カテゴリフォルダで必ず見つけられ
  ます。
