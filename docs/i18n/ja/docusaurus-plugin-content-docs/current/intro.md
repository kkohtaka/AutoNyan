# AutoNyan ユーザードキュメント

このサイトでは、稼働中の AutoNyan を**利用する**方法を説明します。Google
Drive に書類を置き、通知メールを受け取り、分類された結果を確認するまでの
流れを扱います。

:::note

これはサイトの雛形とともに作成されたプレースホルダーページです。実際の
情報設計とコンテンツは後続の作業
（[#339](https://github.com/kkohtaka/AutoNyan/issues/339) および各コンテンツ
Issue）で追加されます。

:::
