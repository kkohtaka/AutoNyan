# AutoNyan

**Google Drive に書類を置くだけで、AutoNyan が自動で整理します。**

AutoNyan は Google Drive のフォルダを監視し、追加された書類を読み取って適切な
カテゴリに分類し、処理内容をまとめたメールを送ります。あなたはこれまでどおり
Drive を使うだけ。整理は AutoNyan がバックグラウンドで行います。

## できること

- **書類を読み取る** — 置かれたファイルから自動でテキストを抽出します。紙の
  書類をスキャンまたは撮影し、PDF として保存したものも対象です。
- **カテゴリに分類する** — 各書類を分類して対応するフォルダへ移動するので、
  手作業で振り分けなくても関連する書類が一か所にまとまります。
- **結果を知らせる** — どの書類が処理され、どこに分類されたかをまとめた通知
  メールが届きます。

## 想定する利用者

このサイトは、**稼働中の AutoNyan を利用する方**に向けたものです。共有された
Drive フォルダに書類を追加し、整理を AutoNyan に任せて使います。基盤となる
クラウドインフラをデプロイ・運用・理解する必要はありません。

:::note

AutoNyan 自体の構築・デプロイ・運用を行いますか？　それはこのサイトではなく、
リポジトリの[開発者向けドキュメント](https://github.com/kkohtaka/AutoNyan#readme)で
説明しています。

:::
