# はじめに

このページでは、AutoNyan を使い始める前に必要なものと、最初の書類の追加方法を
説明します。

## 始める前に

必要なものは 2 つだけです。

- **監視対象の Drive フォルダへのアクセス権。** AutoNyan を設定した人が、
  Google Drive のフォルダをあなたに共有します。これが、処理してほしい書類を置く
  場所です。見つからない場合は、AutoNyan を運用している人にフォルダを共有して
  もらってください。
- **通知用のメールアドレス。** AutoNyan は書類を処理したあと、内容をまとめた
  メールを送ります。これは運用者が設定します。

新しいソフトのインストールや、別サービスへのサインインは不要です。Google Drive
が使えれば AutoNyan も使えます。

## カテゴリを用意する

AutoNyan は各書類をカテゴリに振り分けます。カテゴリとは、Drive のカテゴリ領域内
にあるフォルダのことです。書類を追加する前に、いくつかカテゴリフォルダを用意して
おきましょう。たとえば次のようなものです。

- `請求書`
- `領収書`
- `契約書`

必要なカテゴリごとにフォルダを 1 つ作ります。カテゴリの追加や名前の変更はいつ
でも可能で、次に処理される書類から新しい一覧が使われます。合うカテゴリがない
書類は **未分類（Uncategorized）** に振り分けられます。

## 最初の書類を追加する

1. 共有された監視対象の Drive フォルダを開きます。
2. 書類をドラッグして入れます（いつもどおりアップロードしてもかまいません）。
   AutoNyan が読み取れる形式は[対応ファイル形式](./supported-files.md)を参照して
   ください。
3. 待ちます。AutoNyan はスケジュールに沿ってフォルダを確認するため、処理は即時
   ではありません。少し時間をおいてください。

## 結果が表示される場所

書類が処理されると、

- その書類は監視対象フォルダから**移動**し、該当する**カテゴリフォルダ**（合う
  ものがなければ **未分類**）に入ります。
- 何が処理され、どのカテゴリに振り分けられ、内容がどのようなものだったかを
  まとめた**通知メール**が届きます。

書類が想定より長く監視対象フォルダに残っている場合、たいていは単にタイミングの
問題です。AutoNyan はファイルを置いた瞬間ではなく、スケジュールに沿ってスキャン
します。
